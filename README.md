# yur_Chakra

Jewelry website that displays collections and supports online purchases.  
Clean design, detailed product pages, and a secure checkout system.  
Built with React, TypeScript, and Vite for a fast user experience.

## Development

1. npm install  
2. npm run dev  

## Build

1. npm run build  
2. npm run preview  

## Notes

If you use ESLint with TypeScript, enable type-aware rules and ignore the `dist` folder.
